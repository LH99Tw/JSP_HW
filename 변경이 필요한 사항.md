# 사케 분류 체계 및 명칭 수정 사항

## 1. 사케 분류 체계 수정

### 1.1 정미율(精米歩合) 기준 분류 (Style 필드)

현재 잘못된 분류를 올바른 분류로 수정해야 합니다.

**올바른 분류 체계:**

| 분류명 | 정미율 | 설명 | 비고 |
|--------|--------|------|------|
| **준마이다이긴조** (純米大吟醸) | 50% 이하 | 쌀만 사용, 최고급, 향이 매우 풍부 | 준마이 + 다이긴조 |
| **준마이긴조** (純米吟醸) | 60% 이하 | 쌀만 사용, 향미 강화 | 준마이 + 긴조 |
| **준마이** (純米) | 제한 없음 | 쌀만 사용, 쌀 향이 짙고 구수함 | 알코올 첨가 없음 |
| **혼조죠** (本醸造) | 70% 이하 | 소량의 양조 알코올 첨가, 가벼운 맛 | 알코올 첨가 |

**추가 분류 (제조 방식별):**

| 분류명 | 설명 | 비고 |
|--------|------|------|
| **니고리자케** (濁り酒) | 여과하지 않은 탁주, 흰색 혼탁 | 제조 방식별 |
| **키죠슈** (貴醸酒) | 양조 알코올 대신 사케를 첨가한 고급주 | 제조 방식별 |
| **나마자케** (生酒) | 가열 처리(히오레)를 하지 않은 생주 | 제조 방식별 |
| **나마초조** (生貯蔵酒) | 저장 시 가열하지 않고 병입 시 가열 | 제조 방식별 |
| **나마즈메** (生詰め) | 저장 시 가열하고 병입 시 가열하지 않음 | 제조 방식별 |

**현재 DB의 잘못된 분류:**
- ❌ '다이긴조' (올바름, 유지 가능하나 준마이다이긴조와 구분 필요)
- ✅ '준마이긴조' (올바름, 유지)
- ✅ '준마이' (올바름, 유지)
- ✅ '준마이다이긴조' (올바름, 유지)

**주의사항:**
- '다이긴조'는 혼조조 계열로, '준마이다이긴조'와는 다릅니다 (알코올 첨가 여부 차이)
- 정확한 분류를 위해 제품의 알코올 첨가 여부를 확인해야 합니다

**수정 방안:**
- `style` 필드: 정미율 기준 분류 저장 (준마이다이긴조, 준마이긴조, 준마이, 다이긴조, 긴조, 혼조조)
- 별도 필드(`brewing_method`) 추가 고려: 제조 방식별 분류 (나마자케, 나마초조, 나마즈메, 니고리자케, 키죠슈 등)
- 또는 `style` 필드에 조합하여 저장 (예: "준마이다이긴조 나마자케")

## 2. 브랜드명 및 제품명 수정

### 2.1 현재 문제점

현재 DB의 `name_ko` 필드에 브랜드명만 저장되어 있고, 실제 제품명이 누락되어 있습니다.

**현재 DB 데이터:**
- ❌ '다사이' → ✅ **'닷사이'** (브랜드명)
- ❌ '구보타' → ✅ **'쿠보타'** (브랜드명)
- ❌ '팔해산' → ✅ **'핫카이산'** (브랜드명)
- ❌ '신정' → ✅ **'아라마사'** (브랜드명)
- ❌ '십사대' → ✅ **'쥬욘다이'** (브랜드명)

### 2.2 올바른 제품명 구조

**제품명 = 브랜드명 + 제품명 + 스타일**

예시:
- **닷사이 23 준마이다이긴조** (브랜드: 닷사이, 제품명: 23, 스타일: 준마이다이긴조)
- **아라마사 히노토리** (브랜드: 아라마사, 제품명: 히노토리)
- **쿠보타 만주** (브랜드: 쿠보타, 제품명: 만주)
- **핫카이산 준마이** (브랜드: 핫카이산, 제품명: 준마이)
- **쥬욘다이** (브랜드: 쥬욘다이, 제품명은 다양함)

### 2.3 데이터베이스 스키마 수정 방안

**옵션 1: `name_ko` 필드를 제품명으로 변경**
- `name_ko`: 전체 제품명 (예: "닷사이 23 준마이다이긴조")
- `brand`: 브랜드명 (예: "닷사이")
- `product_name`: 제품명만 (예: "23") - 새 필드 추가

**옵션 2: 현재 구조 유지하되 데이터 수정**
- `name_ko`: 브랜드명 + 제품명 (예: "닷사이 23")
- `brand`: 브랜드명 (예: "닷사이")
- `style`: 스타일 분류 (예: "준마이다이긴조")

**권장: 옵션 2** (기존 스키마 유지하면서 데이터만 수정)

## 3. 구체적인 수정 예시

### 3.1 닷사이 (獺祭, Dassai)

**현재:**
- name_ko: '다사이'
- brand: '獺祭'
- style: '다이긴조'

**수정 후:**
- name_ko: '닷사이 23' (브랜드명 + 제품명)
- brand: '닷사이' (또는 '獺祭')
- style: '준마이다이긴조'
- product_name: '23' (새 필드 추가 시, 또는 name_ko에서 추출)

**참고:** 닷사이는 여러 제품이 있음 (23, 39, 45, 50 등 - 숫자는 정미율을 의미)

### 3.2 쿠보타 (久保田, Kubota)

**현재:**
- name_ko: '구보타'
- brand: '久保田'
- style: '준마이긴조'

**수정 후:**
- name_ko: '쿠보타 만주' (또는 '쿠보타')
- brand: '쿠보타'
- style: '준마이긴조'

### 3.3 핫카이산 (八海山, Hakkaisan)

**현재:**
- name_ko: '팔해산'
- brand: '八海山'
- style: '준마이'

**수정 후:**
- name_ko: '핫카이산 준마이' (또는 '핫카이산')
- brand: '핫카이산'
- style: '준마이'

### 3.4 아라마사 (新政, Aramasa)

**현재:**
- name_ko: '신정'
- brand: '新政'
- style: '준마이다이긴조'

**수정 후:**
- name_ko: '아라마사 히노토리' (또는 '아라마사')
- brand: '아라마사'
- style: '준마이다이긴조'

### 3.5 쥬욘다이 (十四代, Juyondai)

**현재:**
- name_ko: '십사대'
- brand: '十四代'
- style: '다이긴조'

**수정 후:**
- name_ko: '쥬욘다이' (제품명은 다양하므로 브랜드명만 표기하거나 구체적 제품명 필요, 예: '쥬욘다이 나카도리')
- brand: '쥬욘다이'
- style: '준마이다이긴조' (또는 해당 제품의 실제 스타일, 쥬욘다이는 다양한 스타일의 제품이 있음)

## 4. 구현 우선순위

1. **즉시 수정 필요:**
   - 브랜드명 한글 표기 수정 (다사이→닷사이, 구보타→쿠보타 등)
   - style 필드 분류 수정 (다이긴조 → 준마이다이긴조 또는 혼조조 계열로 구분)

2. **단기 개선:**
   - 제품명 필드 추가 또는 name_ko에 제품명 포함
   - 제조 방식별 분류 필드 추가 고려

3. **장기 개선:**
   - 제품명 자동 완성 기능
   - 브랜드별 제품 목록 관리 기능

## 5. 참고 자료

- 일본주도(日本酒度): -3 ~ +10 (음수=단맛, 양수=드라이)
- 정미율(精米歩合): 쌀 겉부분을 깎아낸 비율 (작을수록 고급)
- 산도(酸度): 산미의 강도
- 향 타입: fruity, floral, rice, lactic 등
- 바디감: 1(라이트) ~ 5(풀바디)
