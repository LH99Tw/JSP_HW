  5. 관리자 페이지
  6. 다국어 & UI 다듬기
  7. ML 마이크로서비스 연동
  8. 꾸미기

**언제 보면 좋은가?**
- 전체 개발 흐름을 빠르게 파악할 때
- 개발 우선순위를 확인할 때

### 개발로드맵_세분화.md
**위치**: 프로젝트 루트

- Phase 7: 관리자 페이지
- Phase 8: 다국어 지원 (i18n)
- Phase 9: ML 마이크로서비스 연동
- Phase 10: UI/UX 다듬기 & 최종 통합
- 각 Phase별 체크리스트 포함

**언제 보면 좋은가?**

### 사케도메인지식.md
**위치**: 프로젝트 루트  
**용도**: 사케 도메인 지식 정리

**주요 내용**:
- 사케의 기본 개념
- 사케의 주요 분류
  - 정미율 기준 (다이긴조, 긴조, 준마이 등)
  - 첨가 여부 (준마이, 혼죠조)
- 사케 맛 프로파일 요소
  - 일본주도 (단맛/드라이)
  - 산도
  - 향 (과일향, 꽃향, 곡물향 등)
  - 바디감
- 사케 지역(도도부현) 특성
  - 니가타, 효고, 후쿠시마, 야마가타 등
- 사케 브랜드(주조장) 개념
- 리뷰 시스템에 필요한 도메인 요소
- 사케 상품(커머스) 도메인 요소

**언제 보면 좋은가?**
- DB 스키마를 설계할 때 (필드 의미 이해)
- 검색 필터를 구현할 때
- 리뷰 평가 지표를 설계할 때
- 사케 정보 입력 폼을 만들 때
- 도메인 용어를 확인할 때

### 엔드포인트.md
**위치**: 프로젝트 루트  
**용도**: ML 서비스 API 엔드포인트 목록

- API 엔드포인트를 확인할 때
